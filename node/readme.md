##### What NodeJS Can do

- [ &#x2611; ] Can do Complex Single Page App (SPA's)
- [ &#x2611; ] Real Time Apps (RTA)
- [ &#x2611; ] Chat rooms
- [ &#x2611; ] Browser Games
- [ &#x2611; ] Data Streaming Apps
- [ &#x2611; ] REST API

#### What NodeJS Not Good
- [ &#x2612; ] CRUD App
- [ &#x2612; ] Heavy server-side computation

#### Best Pattern
